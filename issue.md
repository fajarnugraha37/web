# Missing Features in MARKDOWN.EXE

Based on a comparison with [Markdown Viewer](https://markdownviewer.pages.dev/), the following features are missing in the current implementation of our markdown playground:

### 1. Document Management
- [ ] Multiple file support (tabs or sidebar for file switching)
- [ ] File renaming and deletion capabilities
- [ ] Persistent storage (local storage integration)

### 2. Import/Export
- [ ] Import from GitHub
- [ ] Export to HTML
- [ ] Export to PDF

### 3. Toolbar & UI Enhancements
- [ ] Diagram viewer integration (e.g., Mermaid.js)
- [ ] Sync toggle control (manual vs auto-scroll)
- [ ] Dark/Light mode toggle (current is fixed dark)
- [ ] "Copy to Clipboard" / "Copy Share Link" functionality

### 4. Stats/Utility
- [ ] Real-time Read Time calculation
- [ ] Real-time Word Count display
- [ ] Real-time Character Count display

### 5. Editor Features
- [ ] Enhanced toolbar for formatting (bold, italic, lists, etc.)
- [ ] Drag and drop file support
- [ ] Undo/Redo history tracking
